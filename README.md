# Pandas.
Pandas is an open-source library built on top of Python that helps you work with structured data like tables (similar to Excel sheets or databases).
🔹 Why we use Pandas?
We use Pandas to:
Read data from files (Excel, CSV, etc.)
Clean and organize messy data
Analyze data easily
Create summaries and reports
🔹 Main Data Structures in Pandas
Series
A one-dimensional array (like a column)
Example: List of marks
DataFrame
A two-dimensional table (rows + columns)
Like an Excel sheet
